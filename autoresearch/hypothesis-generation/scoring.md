# Scoring: hypothesis-generation
Target: 0.9

1. **Hypothesis Quality (0.30)**: >=3 distinct, non-redundant hypotheses = 1.0; <3 or redundant = 0.5
2. **Scoring Criteria Applied (0.20)**: Each hypothesis scored on testability/falsifiability/parsimony/explanatory power = 1.0
3. **Experimental Tests (0.25)**: Crucial experiments to distinguish hypotheses designed = 1.0; vague tests = 0.5
4. **Evidence Grounding (0.15)**: Hypotheses reference known biology/literature = 1.0; unsupported speculation = 0
5. **Completeness (0.10)**: Required sections present = 1.0

Override: If only 1 hypothesis generated (no competing alternatives), case score = 0.
