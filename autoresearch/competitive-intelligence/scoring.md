# Scoring Checklist: competitive-intelligence

Target: 0.9

## Criteria (per case)

### 1. Landscape Completeness (weight: 0.30)
Are the key approved agents and pipeline competitors identified?
- >= 80% of expected agents identified = 1.0
- 60-79% = 0.5
- < 60% = 0

### 2. Differentiation Analysis (weight: 0.25)
Does the report identify meaningful differentiation axes between competitors?
- Correct differentiation factors from expected list = 1.0
- Partial differentiation analysis = 0.5
- No differentiation or wrong comparisons = 0

### 3. Market Dynamics (weight: 0.20)
Are key market factors captured (LOE, biosimilar impact, safety concerns, cost)?
- Key market dynamics from expected_details documented = 1.0
- Partially addressed = 0.5
- Missing market context = 0

### 4. Pipeline Accuracy (weight: 0.15)
Are pipeline compounds correctly identified with development stage and sponsor?
- Correct phases and sponsors = 1.0
- Correct compounds but wrong stages = 0.5
- Major pipeline misses = 0

### 5. Report Structure (weight: 0.10)
Are required_sections present (landscape matrix, pipeline waterfall, differentiation)?
- All present = 1.0
- Missing 1 = 0.5
- Missing 2+ = 0

## Override
If a major approved agent (e.g., semaglutide in obesity, rituximab in lymphoma) is completely missing from the landscape, deduct 0.3 per miss.
If a failed/withdrawn program is listed as active/approved, deduct 0.2.
